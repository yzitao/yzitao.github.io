Hello World

Posted on 2016-07-23   |   In Personal   |  

####想了解和联系我的可以戳About

这个Hello World还是想要保留下，记录下一个文科生呆呆地、不知所以的建站过程以及自以为是的建站动机。（里面有福利图片！——其实是拿来测试markdown语法的）

时间是2016.7.20日左右

真的是在守望屁股的毒害以及人类原罪之懒惰的影响下，假期过得戾气十足

——或许可以用热力学第二定律的熵增来说明这一自然趋向？（你这文科生看点理科书就拿出来嘚瑟…）

所以呢不知道为何就开始找机会搭建个人网站，先后去看了许多方法吧。

  1. Wordpress-似乎会被墙有时候，太臃肿了，国内的服务也有些蛋疼，要求的技术基础也不少。
  2. Ghost—很喜欢这种非常简约的风格，大抵也都符合我的要求，漂亮–只是似乎14天后就要开始收费了，每个月$14也就算了，对于没有国际交易手段的我就只是忍痛割爱。
  3. Wix，Weebly，Squarespace，Strikingly-大抵是一些大同小异的收费手段，Wix说是拖拽式十分方便，但是依然看得我懵逼，weebly看一个大姐姐用得非常好，但是她说最近经常被墙，主题更新也不勤快了，劝着劝着我也就放弃了不然真的很好看我觉得——lingli姐的网站到时候挂友链。Squarespace死活登不上去，不然很好看似乎——不过前面的大都需要付费。
  4. Medium–我怎么会想到在这上面写东西…
  5. Farbox——国内的一家非常好的博客平台，收费很良心超便宜！许多的主题也很好看，要不是我自己的强迫症导致cho主题的about和相册页面搞不出来，我肯定要用farbox的——最后还是弄出来了，居然是很方便的直接把About.md丢在了文件夹下面，然后直接建个文件夹来放照片就自动有相册了。如果github的服务器不好了就搬过去好了。
  6. Github+Hexo+Next————到了这一次的主角啦。作为之前终端碰都没有碰过的mac使用者来说，这次的博客搭建体验可以说是特别有意思…之前看得主要是在说购买域名，但是自己没有选择购买，大概是觉得现在的github二级域名够用了吧。搭建过程非常蛋疼：$ npm install hexo -g 这个命令通常不能好好运行的，大概是因为没有权限，所以我们需要前面加个sudo，变成$sudo nom install hexo -g但是还是不行的，因为什么源的网速太慢啦，所以需要更换源，更换淘宝源啊什么源似乎都不行，最后找到了一个命令才得以安装：
    $ sudo npm install hexo -g --registry=https://registry.npm.taobao.org
    这个之后大致可以正常的hexo s hexo d -g但是发现了next主题的about页面始终404not found，几近放弃，但是第二天狠下心来全部删除重新下载安装，问题才得到了修复。关于主题的话，自己本来最喜欢Ghost那个的主题，但似乎功能不多，没有category等，最后用了next主题，也很喜欢。

然后呢回到正题

关于这个个人网站嘛

其实呢实际上我是这样想的:

关于未来，我有个梦想（抬头看天，45°状——

话说特别是最近吧，看书看到兴奋处，浮现出莫名的幸福感，大致是生而为人，有幸坐拥人类在短短五千年中创造出无数的精神财富，作为人类精神世界支撑的人文学科、人类非理性的酒神精神以及对于美与灵性追求的艺术学科、人类身为宇宙中体量微小的生物却不屈地去认识自己认识世界的自然科学这三大领域大抵是放置于上方的，大抵为形而上的（把自然科学放在上是因为我切割的不是单纯物质与精神，我看重其中对于世界未知的探索而把其技术部分放置于下）。是故下方就是侧重于技术/物质的计算机技术，经济等社会科学。并不是暴力的分级，而是作为一个我认识世界的一个图景罢了。在不断的学习探索中愈发地对人文、艺术、自然科学感到了深深的敬畏，愿自己能够在不断阅读和学习的过程中窥见终极，不，谈何终极，就光是这些东西带来的乐趣与感动就值得一生追求。————像这样的人不多，金克木老先生是一位，最近看到的物理学家李淼大概也有这样的趋势吧，我还差得远。


不说这些大而化之有的没的了。个人网站建立的目的：

  1. 无论是作为文科生自发的焦虑之感，所谓的输入和输出理论，或是从采铜《精进》一书所说的『最小化可行产品』，自己都要求自己能有足够的输出，尽管或许幼稚可笑。但总得有这样的一个开始。作为输出的平台，zhihu随着粉丝的增多以及平台大牛太多所限，越来越不敢作答了；简书虽好，但总感觉有这样的一个小社群反倒让我感觉静不下心，是故建此站，存储输出用。

  2. 其实最重要的也就是第一点，第二点的话大抵是实用主义。作为能够整合自己的一些思考，甚至后期能够呈现一些作品，无论艺术、游戏、照片，作为简历或者是Portfolio都是比较方便的。

  3. 接着便是个人的惰性，总是把持不住自己沉迷于游戏或者是单纯看书不思考的虚无的满足感之中，如果能有一个比较有个人感觉的站点的话，确实会让我在沉迷游戏之余，写写东西，静下来思考。



大抵就如某位批评家所说的那样：

一位现代生物学家写道：“比起地球上有机生命的历史来，人类区区五万年历史不过象一天二十四小时中的最后两秒。按这个比例，文明的历史只占最后一小时的最后一秒的最后五分之一。”现代作为救世主时代的典范，以一种高度的省略包容了整个人类历史。它同人类在宇宙中的身量恰好一致。

————瓦尔特·本雅明（Walter Benjamin）

作为『现代』的『人』，大抵是渺小而彷徨的。

这就要求我们去做一些挣扎，我觉得，这些挣扎就是把自己对世界的认识，以及所思所想写下来，就是去做一点能够挣脱出人之『异化』之事，完成生而为人的尊严。


以上扯的一堆有的没的，其实我就是想说：


人类很渺小

人类很伟大。

以及

##很高兴见到你：D
